# PC 远程控制 (Mobile PC Remote)

用手机浏览器实时远程控制你的 Windows 电脑 —— 触摸板、键盘、媒体、系统命令、屏幕画面。

## ✨ 功能

| 模块 | 功能 |
|------|------|
| 🖱️ **触摸板** | 滑动移动光标、长按拖拽、双指滚动 |
| ⌨️ **键盘** | 文字输入（支持中文）、快捷键（Ctrl+C/V/Z、Alt+F4 等） |
| 🎵 **媒体** | 音量控制、播放/暂停、上一首/下一首 |
| ⚙️ **系统** | 锁屏、显示桌面、任务视图、开始菜单、休眠 |
| 🖥️ **屏幕** | 实时查看电脑画面（每 1.5 秒刷新，标注鼠标位置） |

## 🏗️ 架构

```
手机浏览器 ←→ 公网(localtunnel) ←→ PC(Python + pyautogui)
```

- **后端**：Python `aiohttp` + `pyautogui` + `Pillow`
- **前端**：单页 HTML/CSS/JS，移动端优化
- **穿透**：`localtunnel` 免费内网穿透，无需公网 IP
- **命令通道**：HTTP POST（可靠）+ WebSocket（低延迟鼠标移动）
- **屏幕流**：定时截图 JPEG 压缩，标注鼠标光标位置

## 🚀 快速开始

### 环境要求

- Windows 10/11
- Python 3.10+
- Node.js（用于 npx 运行 localtunnel）

### 安装依赖

```bash
pip install pyautogui aiohttp pillow
```

### 启动

**方式一：一键启动**

双击 `start.bat`

**方式二：手动启动**

```bash
# 终端 1：启动服务器
python server.py

# 终端 2：启动内网穿透
npx --yes localtunnel --port 5000
```

启动后 localtunnel 会输出一个公网地址（如 `https://xxx.loca.lt`），在手机浏览器打开即可。

### 管理员权限

如果遇到 RDC/UAC 弹窗后无法控制，需要**以管理员身份运行**：
- 右键终端 → 以管理员身份运行 → 重新执行上述命令

## 📱 使用指南

### 触摸板操作

| 手势 | 效果 |
|------|------|
| 单指滑动 | 移动光标 |
| 长按不动 0.5 秒后滑动 | 拖拽（选中文字、拖动窗口等） |
| 双指滑动 | 滚轮滚动 |

### 底部按钮

| 按钮 | 功能 |
|------|------|
| 🔴 按住左键 | 切换式：点一下按住，再点松开（配合滑动=拖拽） |
| 👆 单击 | 鼠标左键单击 |
| 🖱️ 右键 | 鼠标右键单击 |
| 🔁 双击 | 鼠标左键双击 |

### 查看屏幕

底部导航切换到「🖥️ 屏幕」标签，每 1.5 秒自动刷新。红色十字线标注当前鼠标位置。

## ⚠️ 注意事项

1. **localtunnel 验证**：首次打开链接需要输入页面显示的 IP 地址
2. **地址会变**：每次重启 localtunnel 地址会变化，需重新在手机输入
3. **校园网**：如果连不上，可能是网络做了设备隔离，内网穿透方案不受影响
4. **防火墙**：首次运行可能需要允许 Python 通过防火墙

## 🔧 技术细节

### 为什么用 HTTP POST 而不是纯 WebSocket？

localtunnel 等穿透服务对 WebSocket 转发不稳定。高频鼠标移动走 WebSocket 降低延迟，低频操作（点击、按键）走 HTTP POST 保证可靠性。前端做了鼠标移动合并缓冲（16ms 窗口），避免 WebSocket 消息风暴。

### 屏幕传输优化

- 截图缩放到原始尺寸的 50%
- JPEG 质量压缩到 40%
- 仅在切换到「屏幕」标签时传输
- 在截图上绘制光标指示器（红色十字 + 圆圈）
